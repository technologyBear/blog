课程分析

# 计算规则

方案看板相关指标包括：加购、下单和支付，只对“商品定义平台”中招生方案类型的商品进行计算。

# 基础指标

# 1. 访客

对招生方案配置的页面有过访问的用户数量。

# 2. 注册人数

注册人数包括两部分：

  • 注册前当天最后一个页面是招生方案配置的页面，归因于该招生方案。
  • 注册的活动码是该招生方案。

招生方案页面推算人数的计算规则： 当天注册的用户中，注册前最后一个关键页面为当前招生方案，则认为是该招生方案带来的注册人数。

# 3. 加购人数

对招生方案商品有过加购行为的用户数量

# 4. 提交订单人数

对招生方案商品有过下单行为且分摊金额大于0的用户数量。

# 5. 实付成功人数

对招生方案商品有过下单行为且分摊金额大于0的用户数量。

# 6. 支付成功人数

对招生方案商品有过支付成功行为的用户数量（含0元订单）。

# 7. 静默实付人数

  • 静默转化的定义：学员自主下单。
  • 对招生方案商品，学员支付成功且分摊金额大于0的用户数量（含0元订单），简称静默实付人数。

# 8. 静默支付人数

  • 静默转化的定义：学员自主下单。
  • 对招生方案商品，学员支付成功的用户数量（含0元订单），简称静默支付人数。

# 9. 电销实付人数

  • 电销转化的定义：东奥员工（工号属于电销部门的）协助下单。
  • 对招生方案商品，学员支付成功且分摊金额大于0的用户数量，简称电销实付人数。

# 10. 电销支付人数

  • 电销转化的定义：东奥员工（工号属于电销部门的）协助下单。
  • 对招生方案商品，学员支付成功的用户数量（含0元订单），简称电销支付人数。

# 11. 电销的计算规则

支付成功订单中包含工号属于“组织架构为电销部”的订单

# 12. 客服实付人数

  • 客服转化的定义：东奥员工（工号属于客服部门的）协助下单。
  • 对招生方案商品，学员支付成功且分摊金额大于0的用户数量，简称客服实付人数。
